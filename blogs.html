Question-2: What is the difference between innerHTML, innerText, and textContent ?


The difference between these three properties is:
innerHTML: It gives us the inner text including all the tags written within an Element. Such as: Suppose we have an element h1 and inside it we have Hi text. If we write (document.getElementsByTagName('h1').innerHTML) then this will return us Hi with h1 tag.
innerText: It gives us the inner text written within an Element,ignoring the whitespacing. Such as: Suppose we have an element h1 and inside it we have Hi text. If we write (document.getElementsByTagName('h1').innerText) then this will return us Hi,just the text.
textContent: It also gives us the inner text written within an Element but it does not ignore the whitespacing. Such as: Suppose we have an element h1 and inside it we have Hi with a space. If we write (document.getElementsByTagName('h1').textContent) then this will return us Hi ,with a whitespace.

Question-3: What is event delegation in the DOM?


It is a technique where we can add event listerners to the parent element instead of its children elements to avoid the problem of event bubbling.

Question-4: What is event bubbling in the DOM?


It is technique where the program first captures the event from top the root and goes till the targeted element and then starts bubbling up from the targeted element to the root element.

Question-5: How do you create, add, and remove elements using JavaScript?


To create an element we use document.createElement('p'),here p is a paragraph element.
To add an element to the root element or simply add elements , we use document.appendChild('list'),here list is the element we want to add.
To remove an element we use document.removeChild('list'),removing the list element created on the root.
